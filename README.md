# Basic Attandance System App
<mysql, node.js, bootstrap, jQuery>

Import attandencesystem.sql

run "npm i" on command

edit db.js fileaccording to your phpmyadmin username and password

run "node app.js" command

open "localhost:8888"

That's it.

------------------------------------
